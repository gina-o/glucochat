# GlucoChat App README

## Instructions to Run GlucoChat:

1. **Download and Extract the ZIP file:**
   - Download the ZIP file.
   - Right-click the file and select **Extract All...** (Windows) or double-click to unzip it (Mac).
   - Open the extracted folder.

2. **Install Required Dependencies (if applicable):**
   - Ensure you have **Node.js** installed on your computer. You can download it from [Node.js](https://nodejs.org/).
   - Open a terminal (or command prompt) in the folder containing the extracted files.
   - Run the following command to install necessary dependencies:
     ```
     npm install
     ```

3. **Run the App:**
   - Once the dependencies are installed, run the following command to start the app:
     ```
     npm start
     ```
   - The app will open in your default web browser.

4. **Enjoy the App:**
   - The app should now be running locally on your browser, and you can interact with it.

## Additional Notes:
- Ensure **Node.js** is installed and that the dependencies are correctly installed with `npm install`.
- The app uses **React** and will automatically set up a local server when you run `npm start`.
